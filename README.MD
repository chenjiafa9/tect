# My Awesome Game 🎮

一个用 Rust + Bevy 引擎开发的开源 3D 平台游戏！

Tect	源自“Architect”（建筑师），意指构建与创造。

本意为构建一个3D上帝视角的沙盒游戏，操作偏向简单休闲，玩家可以在广阔的地图上自行探索、建造

前期最小实现：一个足够大的场景、角色、可交互物品、角色移动（鼠标右键）、相机控制（旋转、上下缩放）、通用交互机制、引入物理引擎（处理碰撞检测、战斗等）

后续加入建造系统、物品合成系统、战斗系统、增加联机等



## 特性
- 流畅的物理与动画
- 完整的存档系统
- 编辑器友好（规划中）
- 跨平台（Windows/macOS/Linux/Web）
